Site Reliability Engineering

Site Reliability Engineering (SRE) is a discipline that applies software engineering principles to infrastructure and operations problems, aiming to create highly reliable, scalable, and automated software systems. Coined by Google, SRE bridges development and operations, focusing on system availability, performance, monitoring, and reducing manual "toil".

Duties of all SRE

  • Monitoring systems
  • Handling incidents
  • Automation
  • Scaling systems
  • Improving reliability
  • Handling incidents
